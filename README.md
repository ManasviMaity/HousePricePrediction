# House Price Prediction

This project predicts house prices using machine learning techniques.

## Models Used

* Linear Regression
* Random Forest Regressor

## Dataset

Housing Prices Dataset from Kaggle.

## Results

### Linear Regression

* MAE: 970,043
* RMSE: 1,324,507
* R² Score: 0.653

### Random Forest

* MAE: 1,021,546
* RMSE: 1,400,566
* R² Score: 0.612

## Key Findings

* Area was the most influential feature.
* Bathrooms, air conditioning, parking, stories, and bedrooms also had a significant impact on house prices.
* Linear Regression performed better than Random Forest on this dataset.
